# Educational Content Guidelines for ROS 2 Module

## Target Audience
- Beginners to intermediate learners in Robotics and AI
- Computer science students with basic Python knowledge
- Readers new to ROS 2 and humanoid robotics

## Content Principles
- Use simple, clear language
- Provide real-world analogies for complex concepts
- Include step-by-step explanations
- Avoid heavy math or complex theory
- Use consistent terminology throughout

## Teaching Approach
- Concept-first, example-second methodology
- Use the "nervous system" metaphor consistently
- Provide beginner-friendly explanations of technical concepts
- Include common beginner mistakes and misconceptions
- Focus on understanding over implementation complexity

## Technical Accuracy
- All content must align with official ROS 2 documentation
- Verify claims against authoritative sources (ROS 2, OpenAI, etc.)
- Use current LTS version of ROS 2 (Humble Hawksbill)
- Clearly indicate version-specific features

## Content Structure
- Modular chapters that can be consumed independently
- Clear learning objectives at start of each section
- Exercises appropriate for target audience
- Visual diagrams (text-described) to reinforce concepts

## Language Requirements
- Beginner-friendly language (avoid jargon without explanation)
- Consistent terminology across all chapters
- Simple sentence structure
- Active voice preferred
- Positive, encouraging tone

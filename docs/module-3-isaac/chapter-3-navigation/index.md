# Chapter 3: Robot Navigation with Isaac ROS & Nav2

Welcome to the final chapter of Module 3! In this chapter, we'll explore how robots navigate their environment using Visual SLAM (VSLAM), Isaac ROS perception pipelines, and Nav2 path planning for humanoid robots.

## Learning Objectives

By the end of this chapter, you'll be able to:
- Explain Visual SLAM (VSLAM) in simple terms
- Understand Isaac ROS hardware-accelerated perception pipelines
- Describe path planning and navigation for bipedal humanoids
- Explain how Isaac ROS and Nav2 work together
